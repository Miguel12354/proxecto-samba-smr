## Descrición

Este proxecto consiste na implantación dun Controlador de Dominio empregando Samba 4 nun entorno virtualizado.
Permite:
  - Xestión centralizada de usuarios.
  - Unión de equipos Windows ao dominio.
  - Configuración de recursos compartidos.
  - Implementación dun servizo DNS interno.
 ## Obxectivos xerais
- Instalar e configurar Ubuntu Server.
- Configurar Samba como Active Directory Domain Controller.
 -Crear e administrar usuarios e grupos.
- Unir clientes Windows ao dominio.
- Documentar o proceso completo.
 ## Tecnoloxías empregadas

- Ubuntu Server 22.04 LTS ,  Samba 4 , VirtualBox , Windows 10/11 , DNS interno de Samba , Kerberos
